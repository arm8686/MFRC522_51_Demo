# MFRC522_51_Demo
MFRC522 with 51 MCUs, (e.g. STC89, STC12 )

使用 I/O 来模拟 SPI 接口。
在源代码中有更详细的说明。

MFRC522-PA 模块
测试结果：读卡距离 65mm，写卡距离 60mm。
【这两项性能参数同时取决于卡片质量与放置位置】
